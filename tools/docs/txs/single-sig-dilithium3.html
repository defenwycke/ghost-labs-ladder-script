SINGLE SIG (DILITHIUM3)

Single post-quantum signature using Dilithium3 lattice-based cryptography. The simplest possible PQ transaction &mdash; one rung, one SIG block, Dilithium3 scheme. Demonstrates the PQ witness overhead (~3,300 bytes) vs classical Schnorr (64 bytes).

SIG DILITHIUM3 · SCHEME
Rung Structure

Single rung with one SIG block using SCHEME = DILITHIUM3. Identical structure to a classical SIG, with PQ key material.

Each row is a rung (spending path). Rungs are OR — satisfy any one rung to spend. Blocks within a rung are joined by + meaning AND — all must pass.

How Single Sig (Dilithium3) Works
1.
The conditions are identical to a classical SIG rung: one PUBKEY_COMMIT (32 bytes, same as Schnorr). The SCHEME field in the coil selects Dilithium3.
2.
The witness contains the full Dilithium3 public key (~1,952 bytes) and signature (~3,293 bytes). This is ~50x larger than Schnorr (33+64 bytes).
3.
The evaluator verifies SHA256(dilithium3_pubkey) matches the PUBKEY_COMMIT, then validates the Dilithium3 signature against the Ladder sighash.
4.
PQ support is transparent to the condition encoding. The same SIG type code works for all schemes — only the witness and verification algorithm change.
