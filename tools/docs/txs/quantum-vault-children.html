How Quantum Vault + Children Works
1.
The funding TX encodes RECURSE_SPLIT with SCHEME = DILITHIUM3. The split logic is identical to the classical version — two children with decremented max_splits.
2.
Each child inherits the same PQ conditions. The Dilithium3 scheme propagates through the covenant chain automatically.
3.
The recover rung provides a CSV-locked escape path with the recovery key. This works identically to the classical vault recovery.
4.
This preset demonstrates that recursion covenants compose naturally with PQ signatures — no special handling needed.
