SINGLE SIG (FALCON1024)

Single post-quantum signature using FALCON-1024 lattice-based cryptography. NIST Security Level 5 &mdash; the highest tier. Larger signatures (~1,800 bytes) and keys (~1,793 bytes) than FALCON-512, but with a stronger security margin against quantum attack.

SIG FALCON1024 · SCHEME
Rung Structure

Single rung with one SIG block using SCHEME = FALCON1024. Same structure as FALCON-512 but with Level 5 parameters.

Each row is a rung (spending path). Rungs are OR — satisfy any one rung to spend. Blocks within a rung are joined by + meaning AND — all must pass.

How Single Sig (Falcon1024) Works
1.
FALCON-1024 doubles the lattice dimension from FALCON-512, providing NIST Security Level 5 (equivalent to AES-256). Public keys are ~1,793 bytes; signatures are ~1,800 bytes.
2.
The condition encoding is identical to all SIG schemes: a 32-byte PUBKEY_COMMIT in the conditions with FALCON1024 scheme byte in the coil. The evaluator hashes the full PQ public key from the witness and verifies the commitment.
3.
Total witness overhead is ~3,593 bytes (1,793B pubkey + 1,800B signature) — roughly double FALCON-512 but still slightly smaller than Dilithium3's total.
4.
FALCON-1024 uses the same NTRU lattice and Gaussian sampling as FALCON-512 but with larger parameters. Choose FALCON-1024 when maximum PQ security is required and the ~3.6 KB witness overhead is acceptable.
