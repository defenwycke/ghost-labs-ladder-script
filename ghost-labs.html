About

Ghost Labs builds, tests, and proposes new ideas for the Bitcoin protocol. Every project starts as working code on signet and graduates to a formal BIP if the design holds up. Labs projects ship with documentation, test suites, and tooling you can run yourself.

Ladder Script Active -- signet testing

A typed, structured transaction format for Bitcoin. Replaces opcode-based scripting with a declarative block model. 61 block types across 10 families cover signatures, timelocks, hashes, covenants, recursion, anchors, programmable logic, compound patterns, governance, and legacy Bitcoin transaction types. Every byte in a witness is typed. Evaluation is deterministic with bounded execution time.

Native post-quantum support, covenant recursion, and anti-spam hardening via merkle_pub_key and selective inversion.

Process

Every Labs project follows the same pipeline.

1.
Research and prototype
Design the system, write a reference implementation in ghost-core, build comprehensive test coverage. Ship docs alongside code.
2.
Signet testing
Deploy to Bitcoin signet. Build interactive tools so anyone can construct, inspect, and broadcast test transactions.
3.
BIP proposal
Write the formal Bitcoin Improvement Proposal with specification, rationale, security analysis, test vectors, and reference implementation.
